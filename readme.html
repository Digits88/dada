Defense Against the Dark Arts

This repository is for the Defense Against the Dark Arts (CS 4630) course at the University of Virginia. It is a computer science elective taken in the 3rd or 4th year.

The course was originally created by Jack Davidson in 2006 (see a SIGCSE publication on the course). The material in this repository was taken, with permission, from the slides created by Davidson and extended by Charles Reiss and myself, Aaron Bloomfield.

Repository Contents

The "md" links will work if viewed on github.com; the other links work if the repo is cloned locally.

  • Slides (md)
  • UVa specific material (md)

Entering make in the cloned repo directory will convert all the .md files to .html files for local viewing (requires pandoc).
